# Demo Code Snippets

Testing the docs-reviewer with code execution.

## Python Example

```python
# Calculate factorial
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

result = factorial(5)
print(f"Factorial of 5 is: {result}")
```
